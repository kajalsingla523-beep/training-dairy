Day 11 - Java Industrial Training Diary

Date: 5 July 2024

Topic Covered:

Java Packages and Access Modifiers

Description:

Studied Java packages (built-in and user-defined) and how to organize code using them. Understood the four access modifiers in Java and their use in restricting access at class, package, and subclass levels.

Practical Implementation:

Created and used user-defined packages. Explored public, private, protected, and default access modifiers through class structures.

Challenges Faced:

Package import errors due to folder structure. Learned to structure directories properly and compile using `-d` flag.

Conclusion:

Packages help in organizing code efficiently. Access modifiers are crucial for encapsulation and data hiding in Java.
