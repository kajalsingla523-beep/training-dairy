08:00 AM - 10:00 AM

Today we focused on Java Database Connectivity (JDBC) PreparedStatement. We learned why using PreparedStatement is safer and more efficient than Statement, especially in preventing SQL injection attacks.

We created a program that inserts and retrieves employee data from a MySQL database using PreparedStatement.

import java.sql.*;
class InsertExample {
    public static void main(String[] args) {
        try {
            Connection con = DriverManager.getConnection("jdbc:mysql://localhost:3306/company", "root", "password");
            PreparedStatement ps = con.prepareStatement("INSERT INTO employees(name, dept) VALUES(?, ?)");
            ps.setString(1, "Amit");
            ps.setString(2, "IT");
            ps.executeUpdate();
            System.out.println("Record inserted successfully");
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
