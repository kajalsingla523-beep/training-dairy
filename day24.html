We shifted focus to Java Swing for building GUIs. I learned how to create JFrame, add components like JLabel, JButton, JTextField, and register event listeners for user interactions.

import javax.swing.*;
class MyFrame {
    public static void main(String[] args) {
        JFrame f = new JFrame("My First Frame");
        JButton b = new JButton("Click Me");
        b.setBounds(100, 100, 100, 50);
        f.add(b);
        f.setSize(400, 400);
        f.setLayout(null);
        f.setVisible(true);
    }
}
