Day 9 - Java Industrial Training Diary

Date: 3 July 2024

Topic Covered:

Polymorphism in Java

Description:

Studied polymorphism, one of the core concepts of Object-Oriented Programming. Understood the difference between compile-time (method overloading) and runtime polymorphism (method overriding). Explored how dynamic method dispatch works in Java.

Practical Implementation:

Practiced method overriding and dynamic dispatch. Created classes with overridden methods and invoked them through parent class references to observe polymorphic behavior.

Challenges Faced:

Understanding runtime polymorphism was difficult. Initially got confused between overloading and overriding, but examples helped clarify.

Conclusion:

Polymorphism improves code flexibility and reusability. I now feel confident in implementing method overloading and overriding in Java.
