08:00 AM - 10:00 AM

We began learning about Java Exception Handling. The trainer explained the difference between checked, unchecked, and errors in Java. We explored common exceptions like NullPointerException, ArrayIndexOutOfBoundsException, and IOException.

We also studied the try, catch, finally, throw, and throws keywords.

public class Example {
    public static void main(String[] args) {
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("Cannot divide by zero!");
        } finally {
            System.out.println("Execution completed.");
        }
    }
}
