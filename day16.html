I learned how interfaces help achieve multiple inheritance in Java, which is otherwise not possible using classes alone. The trainer explained the syntax, the implements keyword, and scenarios where interfaces are the preferred choice.

We also explored default methods in interfaces introduced in Java 8, which allow us to add new methods without breaking existing implementations.

Example discussed in class:

interface Animal {
    void sound();
    default void eat() {
        System.out.println("Eating...");
    }
}

class Dog implements Animal {
    public void sound() {
        System.out.println("Bark");
    }
}

public class TestInterface {
    public static void main(String[] args) {
        Dog d = new Dog();
        d.sound();
        d.eat();
    }
}
