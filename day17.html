Day 17 - 16 July 2024

Java Development Industrial Training

08:00 AM - 10:00 AM

Today’s focus was on the Java Collections Framework. The trainer explained the main interfaces: List, Set, and Map, and their most common implementations like ArrayList, LinkedList, HashSet, TreeSet, HashMap, and TreeMap.

We discussed why collections are more efficient and flexible compared to arrays, especially in real-world applications where dynamic data structures are needed.

import java.util.*;
public class ListExample {
    public static void main(String[] args) {
        List names = new ArrayList<>();
        names.add("John");
        names.add("Alice");
        names.add("Bob");
        System.out.println(names);
    }
}

10:00 AM - 12:00 PM

We practiced several programs:

  • Storing and retrieving data from a List.
  • Removing duplicates using a Set.
  • Mapping student IDs to names using a Map.

We also covered iterating over collections using for-each loops, Iterator, and Java 8’s forEach() method with lambdas.
