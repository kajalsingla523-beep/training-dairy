Java Development Industrial Training

08:00 AM - 10:00 AM

The day started with learning about sorting collections. We studied the Collections.sort() method for Lists, and how natural ordering is determined via the Comparable interface.

We then explored the Comparator interface for custom sorting, such as sorting students by marks, names, or IDs.

import java.util.*;
class Student implements Comparable {
    String name; int marks;
    Student(String n, int m) { name = n; marks = m; }
    public int compareTo(Student s) { return this.marks - s.marks; }
}
public class SortExample {
    public static void main(String[] args) {
        List list = new ArrayList<>();
        list.add(new Student("Alice", 85));
        list.add(new Student("Bob", 92));
        Collections.sort(list);
    }
}
